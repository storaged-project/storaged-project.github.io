Creates a new volume from devices.

No additional options are currently defined.

Changed in version 2.7.0.

IN ao devices:

An array of object paths to objects implementing the #org.freedesktop.UDisks2.Block interface.

IN s label:

The label for the new volume.

IN s data_level:

The RAID level for the new volume.

IN s md_level:

The RAID level for metadata.

Additional options.
