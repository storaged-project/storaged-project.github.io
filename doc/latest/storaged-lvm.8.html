storaged-lvm — The storaged command line tool

SYNOPSIS

storaged-lvm [-b] [-f] COMMAND

DESCRIPTION

storaged-lvm is a helper program used by Storaged to query the state of LVM2 via the lvm2app library.

OPTIONS

-h

Show help options.

-b

Print the result of COMMAND in binary form (serialized).

-f

Use an alternative LVM system directory; no lock.
