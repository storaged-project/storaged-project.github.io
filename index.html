About Storaged

The Storaged project provides a daemon, tools and libraries to access and manipulate disks and storage devices.

The code has its origins in the udisk project which is being developed for the GNOME desktop environment needs. The focus of the Storaged is the D-Bus API suitable for wider audience than just the desktop applications.

Udisks and storaged projects merged in February 2017 using the original name udisks.
