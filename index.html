About Storaged

The Storaged project provides a daemon, tools and libraries to access and manipulate disks and storage devices.

The code has its origins in the udisk2 project which is being developed for the GNOME desktop environment needs. The focus of the Storaged is the D-Bus API suitable for wider audience than just the desktop applications.

There are two variants of the API provided by Storaged: one is Storaged-specific meant to be parallel-installable with udisks2 and second that aims to be complete drop-in replacement for udisks2 providing all the features of udisks2 together with the new modules.
